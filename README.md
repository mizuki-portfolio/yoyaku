# 体育館予約システム

体育館のコート予約を管理するWebアプリケーションです。使用責任者のログイン機能付きで、予約の作成・確認・キャンセルができます。

## 機能

### 認証機能
- 使用責任者の新規登録・ログイン
- セッション管理によるログイン状態の維持
- ログアウト機能

### 予約機能
- 日付を選択して予約可能な時間帯を表示
- Aコート・Bコートの時間別予約
- 利用目的の選択（大会/大会以外）
- 使用責任者名の自動入力（登録者反映ボタン）
- 予約内容の確認画面
- 一度予約された時間帯は選択不可

### 料金計算
- 1時間あたり500円で自動計算
- コートごとの使用時間を集計
- 合計料金の表示
- 予約時間が選択されているときのみ表示

### 予約確認・管理
- 自分の予約のみ表示（他の使用責任者の予約は非表示）
- 予約一覧の確認
- 予約のキャンセル（自分の予約のみ）
- 時間選択画面でのキャンセルは不可（予約確認ページからのみ可能）

## 使い方

### 1. ログイン
1. `login.html`をブラウザで開く
2. デモアカウント（山田太郎/0000 または 佐藤花子/0000）でログイン
3. 新規登録の場合は「新規登録」ボタンからアカウントを作成

### 2. 予約入力
1. 利用目的を選択（大会/大会以外）
2. 利用目的詳細を入力
3. 予約日を選択
4. 予約したい時間帯のコートをクリック
5. 「登録者反映」ボタンで使用責任者名を自動入力
6. 使用人数を入力
7. 使用料金を確認
8. 「確認」ボタンで予約内容を確認
9. 「送信」ボタンで予約を確定

### 3. 予約確認・キャンセル
1. ヘッダーのハンバーガーメニューから「予約確認ページへ」を選択
2. 自分の予約一覧が表示されます
3. キャンセルしたい予約の「キャンセル」ボタンをクリック

## 技術仕様

### データ保存
- **ユーザー情報**: localStorage（`users`キー）
- **予約データ**: localStorage（`reservation_日付`キー）
- **ログイン状態**: sessionStorage（`loggedInUser`キー）

### 料金計算
- 基本料金: 500円/時間
- コートごとに時間を集計
- 合計料金を自動計算

### レスポンシブデザイン
- デスクトップ: 横並びレイアウト
- タブレット・スマホ: 縦並びレイアウト
- メディアクエリで対応（768px以下がモバイル）

## 注意事項

- ブラウザのlocalStorageを使用するため、ブラウザをクリアするとデータが削除されます
- セッション管理のため、ブラウザを閉じるとログアウトします
- 一度予約が確定された時間帯は、予約確認ページからのみキャンセル可能です
- 他の使用責任者の予約は表示・操作できません
